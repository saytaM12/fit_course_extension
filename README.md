# Browser extension for course registration on BUT STUDIS

## Features
- Filtering courses
- Hiding full courses
- Sorting based on any column
- Showing only the relevant semester

## Usage
### Form on top of the page
- Switching semesters
- Hiding full courses
- Filtering
### Clicking the table headers
- Sorting courses
### "Show additional info" button
- Show information about the study program (hidden by default)
